# Мультиагентная система анализа карьерного рынка IT

Мультиагентная система для генерации структурированных карьерных отчётов с использованием YandexGPT.  
Система принимает название IT‑специальности и возвращает детальный отчёт в форматах JSON и Markdown, содержащий навыки, зарплатные вилки, план обучения, анализ пробелов, идею портфолио‑проекта и итоговую проверку отчета.

# Установка и запуск

### 1. Клонирование репозитория
```git clone https://github.com/boushoon/raft-test-task.git```

### 2. Создание и активация виртуального окружения
- ```python -m venv venv``` - создание

- ```venv\Scripts\Activate.ps1``` - активация

### 3. Установка зависимостей
```pip install -r requirements.txt```

### 4. Настройка переменных окружения
- ```cp .env.example .env``` - копирование из файла-примера

- Откройте ```.env``` и заполните нужные данные

### 5. Команда запуска
```python main.py --role "...your_role..."```

### 6. Результаты
После выполнения отчеты и логи(```execution.log```) будут лежать в директории ```reports/```


# Описание агентов

## 1. MarketAnalyst - Аналитик рынка
- Вход: 
  - ```role``` - Название специальности
- Выход: 
  - ```skill_map``` - Словарь навыков с востребованностью и трендом

## 2. SalaryAppraiser - Оценщик зарплат
- Вход:
  - ```skill_map``` - Словарь навыков с востребованностью и трендом
- Выход: 
  - ```salary_table``` - Таблица вилок по грейдам и регионам
  - ```market_trend``` - Тренд рынка
  - ```matker_trend_reason``` - Короткое обоснование тренда рынка
  - ```top_employers``` - Несколько компаний, которые ищут сотрудника с этой специальностью

## 3. CareerAdvisor - Карьерный советник
- Вход:  
  - ```skill_map``` - Словарь навыков с востребованностью и трендом
  - ```salary_table``` - Таблица вилок по грейдам и регионам
  - ```market_trend``` - Тренд рынка
  - ```matker_trend_reason``` - Короткое обоснование тренда рынка
  - ```top_employers``` - Несколько компаний, которые ищут сотрудника
- Выход:
  - ```learning_path``` - Пошаговый план обучения(3 фазы по 30 дней), с темами, ресурсами и ожидаемыми результатами
  - ```gap_analysis``` - Выявление пробелов в навыках, что можно изучить быстро, а на что потребуются месяцы
  - ```portfolio_project``` - Конкретная идея проекта для демонстрации навыков с описанием и списком используемых технологий

## 4. Critic - Критик и верификатор
- Вход: 
  - Полный JSON от агентов 1-3
- Выход:
  - ```quality_score``` - Целочисленная оценка качества отчёта от 0 до 100
  - ```quality_reason``` - Краткое текстовое обоснование, почему выставлена именно такая оценка
  - ```warnings``` - Массив предупреждений о найденных несоответствиях или проблемах
  - ```is_consistent``` - Булевое значение, показывающее, является ли отчёт целостным
